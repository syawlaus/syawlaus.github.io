Leetcode 题解 - 242. Valid Anagram

发表于 2016-06-29   |   分类于 Leetcode

#题目

给定两个字符串 s 和 t，编写一个函数，判断 t 是否 s 的易位构词。

示例：

s = “anagram”，t = “nagaram”，返回 true。
s = “rat”，t = “car”，返回 false。

注意：

你可以假定字符串只包括小写字母。

后续：

如果输入包含 Unicode 字符，你的程序要怎么处理？

难度：容易

编程语言：C++

#分析

程序框架为：

首先解释一下所谓的 anagram（易位构词）。如果 t 是 s 的（其中一个）易位构词，即是使用 s 中每一个字符（且使用一次）构成一个新词 t。

一个简单的思路是，遍历 t 的每个字符 ct，如果在 s 找到 ct，就标记为“已使用”，在遍历 t 完成后，如果“已使用”的标记个数等于 s 的字符数目，则返回 true。

在遍历前先进行剪枝，如果 t 的大小不等于 s 的大小，那么 t 肯定不是 s 的易位构词。
