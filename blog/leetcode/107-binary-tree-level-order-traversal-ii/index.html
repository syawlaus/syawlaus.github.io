Leetcode 题解 - 107. Binary Tree Level Order Traversal II

发表于 2016-10-13   |   分类于 Leetcode

#题目

给定一棵二叉树，对于节点值，返回自底向上的层次顺序（bottom-up level order）的遍历结果（即是，从左到右，从叶子到根节点，一层接一层）。

示例：

给定二叉树 [3, 9, 20, null, null, 15, 7]，

  3
 / \
9  20
  /  \
 15   7

返回自底向上层次顺序的遍历结果：

[
  [15,7],
  [9,20],
  [3]
]

难度：容易

编程语言：C++
